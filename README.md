# PakFreelance

> A localized, AI-assisted freelancing platform purpose-built for Pakistan.

## Overview

Global platforms like Fiverr and Upwork operate under financial structures that systematically disadvantage Pakistani professionals through high fees, mandatory payment cooldowns, and lack of domestic payment support. 

**PakFreelance** solves this by offering a platform uniquely designed for Pakistan featuring domestic payments, enforced escrow, intelligent planning tools, and verified identities on both sides of every contract.

## Key Features

1. **Milestone-Based Escrow System** - Client funds are deposited per milestone and held until delivery is verified.
2. **KYC Verification** - CNIC-based identity checks via Sumsub.
3. **Local Payment Integration** - Zindagi wallet and direct bank transfers (0 currency conversion delays).
4. **Dispute Resolution** - Automated resolution recommendations governed by platform admins.
5. **AI Project Intelligence Engine** 
   * Requirement Expansion
   * Milestone Generation
   * Sprint Planning
   * COCOMO-II Estimator (Effort/Cost/Schedule Prediction)
   * Risk Analyzer
6. **Secure Real-Time Messaging** - WebSocket-based chat for contracts and teams.

## Tech Stack

* **Frontend:** Next.js (v16), React
* **Backend:** Node.js (v22.13), REST APIs
* **Database:** MongoDB (v8.2), Mongoose ORM
* **Real-time:** Socket.io
* **Payments & Integrations:** Zindagi API, Stripe API, Sumsub SDK
* **AI Ecosystem:** OpenAI API
* **Deployment:** Vercel
